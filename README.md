# New_Todo

##  Restart Todo Project

## version 1.0.3

### 新增功能

- 添加待办预计完成时间字段,字段格式为yyyy-mm-dd

## version 1.0.2

### 新增功能

- 更新采用Flex布局
- 更新使用原生js,代替jQuery的使用.主要包括DOM操作和事件操作

### 已知bug

-  绑定和触发事件采用的标准浏览器方式
-  使用非IE(包括Microsoft Eadge)均可正常使用,IE上无法使用.

## version 1.0.1

### 新增功能

- 状态的筛选
  - 已完成
  - 未完成
  - 全部
- 输入检验
  - 如果输入为空,显示"输入不能为空",持续2s.

## version 1.0.0

### 基本功能

- 进入网页自动加载已有待办项
- 自由添加待办项
- 储存到本地PouchDB中, 添加字段包括
  - _id: 添加时间的IOS格式作为唯一关键字
  - title: 输入待办字符串
  - date: 添加时间的"年-月-日"格式
- 修改待办项状态
  - 完成状态的修改
  - 是否删除该待办
